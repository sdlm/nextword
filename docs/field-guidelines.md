# Гайдлайны по заполнению полей

Требования к содержимому каждого поля карточки. Язык полей — английский, кроме `Translation` (русский).

## Word

Слово в начальной форме: глагол — инфинитив, существительное — единственное число, прилагательное — без степеней сравнения.

## Part of speech

Одно из: `verb`, `noun`, `adjective`, `adverb`, `preposition`, `conjunction`, `phrase`.

## Definition

Краткое определение на английском. Одно-два предложения, без примеров.

## Example

Список примеров: 2–3 простых коротких предложения + 1–2 предложения уровня C1 (сложная структура, академический/деловой контекст). Каждый пример — отдельный элемент списка. Текущее слово выделяется **жирным** (bold).

Пример:
```
- She **undertook** the task without hesitation.
- They **undertook** a long journey across the country.
- The government **undertook** a comprehensive review of existing legislation to address emerging regulatory challenges.
```

## Translation

Список вариантов перевода на русский, сгруппированных по смыслу (синонимичные варианты объединяются в один пункт). Каждый вариант — отдельный элемент списка. Для каждого варианта указывается примерная частотность использования.

Пример:
```
- предпринимать, браться за — 60%
- обязываться, брать на себя — 30%
- гарантировать (устар.) — 10%
```

## Collocations

Список устойчивых сочетаний. Каждый пункт списка — группа близких сочетаний, внутри которой варианты перечисляются через ` / `. Слово в сочетаниях выделяется **жирным**.

Пример:
```
- **undertake** a task / project / review
- **undertake** a responsibility / commitment
- **undertake** a journey / expedition
```

## Synonyms & Nuance

**Опциональное поле.** Заполняется только если есть значимые синонимы. Если синонимов нет — поле оставляется пустым.

Список синонимов. Для каждого синонима указывается, чем он отличается от текущего слова.

Пример:
```
- take on — less formal, everyday usage
- embark on — emphasizes the beginning of something large-scale
- assume — specifically for duties, roles, and responsibilities
```

## Cloze

Одно предложение, в котором текущее слово заменено на `...`. Предложение должно быть таким, чтобы пропущенное слово было характерным и угадываемым по контексту.

Пример:
```
The company ... a major restructuring of its operations.
```
